Add a small version of the Jigoshop cart to your shop.

== Description ==

This plugin adds a small version of the Jigoshop cart to your shop. 

No need for widgets or anything like that. Just install, and you are good to go!

Note: This plugin requires that you have Jigoshop installed.

== Installation ==

1. Upload the folder `jigoshop-mini-cart` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Can I modify the CSS? =

Yes, of course. Simply create a CSS-file in your theme folder and name it "minicart.css". The plugin will automatically look for this file before loading its own css.
